Write:
write(1, "texto", 1)	=>	o segundo 1 é o numero de caracteres que vc quer que apareça, se deixar 1, vai aparecer só o "t" nesse caso.
colocar uma variável no write:

x = '1';
write(1, &1, 1"

transformar uma variável em caracter asc para ser lido pelo write:
x = 1 + 1;
Se vc quiser que o write leia que é 2, tem que colocar:
x = (1 + 1) + '0';
write (1, &x ,1) => vai imprimir 2.
